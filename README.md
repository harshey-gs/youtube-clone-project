# 🎬 YouTube Clone Project

A front-end clone of the popular video streaming platform YouTube, built using HTML and CSS. This project replicates the layout and design of YouTube’s homepage for learning and practice purposes.

---

## 🚀 Features

- 🎥 YouTube-like homepage UI
- 📺 Video thumbnails with titles and channel info
- 🔍 Search bar (UI)
- 📂 Sidebar navigation menu
- 📱 Responsive layout (basic)
- 🎨 Clean and modern design using CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
