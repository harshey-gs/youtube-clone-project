🎬 YouTube Clone Project

A front-end clone of the YouTube homepage built using HTML and CSS, replicating the layout, design, and basic structure of the popular video streaming platform.

📌 Project Overview

This project is a YouTube UI clone created for learning and practicing front-end development skills.
It focuses on recreating the visual layout and styling of YouTube using modern CSS techniques like Flexbox and Grid.

🚀 Features
📺 YouTube-like homepage layout
🔍 Search bar (UI only)
🎞️ Video thumbnails with titles and channel info
📂 Sidebar navigation (Home, Subscriptions, Library, etc.)
👤 Channel profile section
🎨 Clean and responsive design
🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
Flexbox & Grid – Responsive design
📁 Folder Structure

💻 How to Run the Project
Clone the repository:
git clone https://github.com/harshey-gs/youtube-clone-project.git
Open the folder:
cd youtube-clone-project
Run the project:
Open youtube.html in your browser
