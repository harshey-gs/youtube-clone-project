# 🎬 YouTube Clone Project

A front-end clone of YouTube built using HTML and CSS. This project replicates the homepage layout of YouTube for learning and practice purposes.

---

## 🚀 Features
- YouTube-like homepage UI
- Video thumbnails with titles and channel info
- Search bar (UI)
- Sidebar navigation menu
- Clean and modern design

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid

---

## 📁 Project Structure
youtube-clone-project/
│── youtube.html
│── youtube-styles/
│── icons/
│── library-icons/
│── thumbnails/
│── channel-profile/

---

## ⚙️ How to Run
1. Clone the repository:
   git clone https://github.com/harshey-gs/youtube-clone-project.git

2. Open the folder:
   cd youtube-clone-project

3. Run the project:
   Open youtube.html in your browser

This project is created to practice front-end development and understand layout design using CSS.
